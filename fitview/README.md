# fitview

Lets you view the summary metadata for .fit, .gpx, and .tcx fles.

## Usage

```text
Usage: fitview [OPTIONS] <FILE(S)>...

Arguments:
  <FILE(S)>...  One or more .fit, .gpx or .tcx file(s) to process. Wildcards and multiple_occurrences files (e.g. 2019*.fit 2020*.gpx) are supported.

Options:
  -s, --print-summary  Print a summary of the number of files processed, errors, etc.
  -l, --print-detail   Print more detail for each file processed.
  -h, --help           Print help (see more with '--help')
  -V, --version        Print version
```
